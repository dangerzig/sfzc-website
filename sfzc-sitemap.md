## URL PATTERN NOTES

1. **Dual path structure**: Many pages are accessible via both `/locations/{center}/...` and `/practice-centers/{center}/...` paths. The `/practice-centers/` prefix appears to be the current canonical pattern, with `/locations/` as a legacy or redirect path.

2. **Calendar events**: Follow the pattern `/calendar/events/{location}/{event-title-slug}`

3. **Dharma talks**: Follow the pattern `/teachings/dharma-talks/{talk-slug}`

4. **Teacher profiles**: Follow the pattern `/teachers/{teacher-slug}`

5. **Files/documents**: Follow the pattern `/files/{document-slug}`

6. **The site has no XML sitemap** (both /sitemap.xml and /sitemap_index.xml return 404).

7. **Drupal CMS**: Confirmed by robots.txt patterns (e.g., /node/add/, /admin/, /filter/tips).
